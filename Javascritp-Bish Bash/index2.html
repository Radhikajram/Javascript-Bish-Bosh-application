Bish Bosh - v2.0


Bish-Bosh application prints range of numbers,inputed through the field Loop Through util filed.

If the number is divisible by 'BishInput' it prints the word 'Bish' not the actual number.

If the number is divisible by 'BoshInput' it prints the word 'Bosh' not the actual number.

If the number is divisible by both 'BishInput' and 'BoshInput' it prints the word 'Bish-Bosh'.

The input is restricted to only integer numbers not float. Also we cannot enter any non-numeric values.
